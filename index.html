// sobre mim

Quem sou eu

Sou um desenvolvedor self-taught que aprende construindo coisas reais. Comecei por curiosidade, e hoje transformo ideias em software que funciona — com Python, FastAPI, automação e análise de dados.

No dia a dia, gerencio 3 lojas do grupo O Boticário. Foi lá que nasceu minha obsessão por automação: cada processo manual repetitivo virou um projeto de código. Extrair estoque do ERP, gerar relatórios, analisar performance de vendedoras — tudo automatizado.

Estou retomando a graduação em Ciência da Computação, combinando a base acadêmica com a visão de quem já coloca código em produção. Aprendizado contínuo não é só um clichê — é meu modo de operação.

"O melhor código é o que resolve o problema real de alguém — de preferência, de forma automática."

SKUmetrics V5.1

Plataforma completa de métricas de SKU para varejo. Backend assíncrono com FastAPI + PostgreSQL + SQLAlchemy 2.x, 72 testes pytest, CI/CD com GitHub Actions, Docker multi-stage, WebSocket em tempo real, Curva ABC, GMV, CMV ponderado e importação assíncrona de planilhas Excel.

FastAPI PostgreSQL Docker GitHub Actions Alembic asyncpg Locust 72 testes

Automação Estoque SGI

Extrai dados de estoque do ERP SGI (O Boticário) via Playwright e sincroniza automaticamente com Google Sheets. Roda diariamente em múltiplas lojas, sem intervenção humana.

Python Playwright gspread Google Sheets API
📊

Análise de Vendedoras

Dashboard HTML + JS puro para análise de performance de vendedoras em lojas O Boticário. Curva ABC por ciclo, ticket médio, risco de estoque e ranking multimarca. 108 testes automatizados com Node.js.

HTML JavaScript Chart.js 108 testes

Power BI — Painel de Entregas

Dashboard de controle de entregas em formato PBIP com DAX avançado, schema estrela otimizado, métricas de SLA e drill-through por região e período.

Power BI DAX PBIP Star Schema

Dashboard Autostart

Sistema que abre dashboards Streamlit e Power BI automaticamente ao ligar o Windows, usando Playwright assíncrono com agendamento via Task Scheduler.
